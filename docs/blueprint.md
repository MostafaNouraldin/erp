# **App Name**: Al-Mustaqbal ERP

## Core Features:

- Multilingual Support: Enable users to switch between the Arabic and English language seamlessly to cater to diverse user preferences and international branches.
- Role-Based Permissions: Provide role-based access control to ensure that each user can only access the modules and functions relevant to their role, enhancing security and compliance.
- Audit Logging: Implement a centralized logging system to track user actions and system events for auditing and troubleshooting purposes.

## Style Guidelines:

- Primary color: Neutral beige (#F5F5DC) to create a calm and professional environment.
- Secondary color: Light gray (#E0E0E0) for backgrounds and subtle UI elements.
- Accent: Teal (#008080) to highlight key actions and important information.
- Clean and professional sans-serif fonts for clear readability in both Arabic and English.
- Use a grid-based layout for consistent alignment and responsive design across different screen sizes.
- Use a consistent set of minimalist icons to represent different modules and functions.
